# soajs.multitenant
This is the soajs multi tenant service 

### Environment variables

ENV Variable | Description | Default | Example
--- | ----- | :---: | ---
SOAJS_SERVICE_MODEL | The model | mongo | other models can be implemented aka oracle, postgresql
